Running locally
---

On a mac, clone the repo, and run a server:

```
$ git clone https://github.com/NathanielWroblewski/de-bruijn-projections.git
$ cd de-bruijn-projections
$ open http://localhost:8000 && python -m SimpleHTTPServer
```
